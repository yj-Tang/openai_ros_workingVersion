## Install Dependencies

Execute the following commands:<br>
`cd ~/ros_ws/src`<br>
`git@github.com:yj-Tang/openai_ros_workingVersion.git`<br>
`cd ~/ros_ws`<br>
`catkin_make`<br>
`source devel/setup.bash`<br>
`rosdep install openai_ros`<br>
